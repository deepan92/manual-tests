Issue 1241: Highlighting of changed CSS

Change class

Steps to reproduce

  1. Open Firebug
  2. Switch to the HTML panel and there to the Style side panel
  3. Inspect the blue <div> above
  4. Click the Change class button

Expected result

  • Inside the Style side panel the class1 rule should be replaced by class2 the whole rule should be highlighted.

Change rule

Steps to reproduce

  1. Open Firebug
  2. Switch to the HTML panel and there to the Style side panel
  3. Inspect the blue <div> above
  4. Click the Change rule button

Expected result

  • Inside the Style side panel the value of the width property inside the #element rule should be changed to 200px and the value should be highlighted.
  • An (overwritten) color property with the value white should be added inside the #element rule and the property incl. its value should be highlighted.

Change inline style

Steps to reproduce

  1. Open Firebug
  2. Switch to the HTML panel and there to the Style side panel
  3. Inspect the blue <div> above
  4. Click the Change inline style button

Expected result

  • Inside the Style side panel the element.style rule should be added including a background property with the value #FFFF8C -moz-linear-gradient(135deg, #FFFF8C, #FFFFC8) should be added inside the element.style rule and the property incl. its value should be highlighted.
